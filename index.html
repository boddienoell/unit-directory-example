Boddie Noell Enterprises Example Unit Directory

This is a public version of the Unit Directory currently under development. An demo of this code is found at here

The web app will serve as our master directory for our employees in the field and in the corporate office. More information and details about the project will be published as time goes on. We are providing this code to the general public in hopes that it will do the following.

  • Provide an example of the type of projects that are under development at Boddie Noell
  • Feedback about what can be code can be improved with the code
  • A working example that someone may use to help them understand Angular and Material Design
